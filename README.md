# E-commerce Website

A full-stack e-commerce website built using PHP, MySQL, HTML, CSS, and JavaScript.  

## Features
- User authentication (login, register, logout)
- Admin dashboard to manage products
- Shopping cart functionality
- Product pages and categories

## Tech Stack
- PHP, MySQL, HTML, CSS, JavaScript
- XAMPP for local server
- Git & GitHub for version control

## How to Run Locally
1. Clone the repo: `git clone https://github.com/Harisaisri/ecommerce-website.git`
2. Copy files to `C:/xampp/htdocs/`
3. Import `database.sql` into phpMyAdmin
4. Update `includes/db.php` with your local database credentials
5. Open `http://localhost/ecommerce-website/` in browser
